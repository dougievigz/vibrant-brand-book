# Colors

## Core Color Palette

The core palette is neutral and sophisticated, designed to let product and art direction take center stage. Reflex Blue, Lab Coat White, and Ink Black are the three primary brand colors.

### Reflex Blue
- **Hex:** `#0142E2`
- **RGB:** 1, 66, 226
- **CMYK:** 100 / 71 / 0 / 11
- **Pantone:** 2728 C
- **Usage:** Primary brand color, CTAs, accents, on-color backgrounds
- **Accessibility:** White text on Reflex Blue = AAA

### Lab Coat White
- **Hex:** `#F3F3F3`
- **RGB:** 243, 243, 243
- **CMYK:** 0 / 0 / 0 / 5
- **Pantone:** White C
- **Usage:** Primary background, light canvas
- **Accessibility:** Black text on Lab Coat White = AAA

### Ink Black
- **Hex:** `#161616`
- **RGB:** 22, 22, 22
- **CMYK:** 0 / 0 / 0 / 91
- **Pantone:** Black C
- **Usage:** Primary text color, dark backgrounds, logo on light surfaces
- **Accessibility:** White text on Ink Black = accessible
- **Note:** Never use #000000 pure black. Always use #161616.

### Silver Foil
- Metallic specialty treatment. No digital hex value.
- Used for premium print and physical applications.

## Color Usage Rules

### Do
- Use core palette (Reflex Blue, Lab Coat White, Ink Black) for all primary communications
- Use Zoomer colors as accent/category colors tied to specific products
- Choose HEX for digital, CMYK for print, Pantone when working with print vendors
- Maintain approved text/background pairings for accessibility

### Do Not
- Never use pure black (`#000000`) - always use Ink Black (`#161616`)
- Never use Zoomer colors as general brand colors outside of product context
- Never use unapproved color combinations
