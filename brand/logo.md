# Logo

## Logo System Components

The Vibrant logo system has three components used independently and in combination.

### 1. Primary Logo (Logomark)
Brandmark + Wordmark together, horizontally arranged. Brandmark to the left, "VIBRANT" wordmark to the right.

**File:** `assets/logos/Vibrant-Primary-Logo-Ink.png`
**File (Lab Coat variant):** `assets/logos/Vibrant-Primary-Logo-Lab-Coat.png`

### 2. Brandmark (Icon Only)
Organic, molecular/biological symbol with rounded forms - interconnected bulbous shapes suggesting cellular or molecular structures.

**File:** `assets/logos/Vibrant-Brandmark-Ink.png`

### 3. Wordmark (Text Only)
"VIBRANT" in all-caps geometric sans-serif with wide tracking.

**File:** `assets/logos/Vibrant-Wordmark-Ink.png`

**Full logo ZIP packages:** `downloads/Primary-Logo.zip` | `downloads/Supporting-Logos.zip`

## Clearspace

### Primary Logo (Logomark)
- Minimum clearspace: **50px on all four sides** (measured as brandmark height = 50px)
- Nothing may appear within this zone: text, images, other logos, decorative elements

### Brandmark (Icon)
- Clearspace measured as the height of the capital "T" in the wordmark

### Wordmark
- Clearspace: width of the brandmark at 50px on all sides

## Logo Placement

### 9-Point Grid (Portrait/Vertical)
The logo can be placed at any of 9 points in a 3x3 grid:

```
01 (top-left)    04 (top-center)    07 (top-right)
02 (mid-left)    05 (mid-center)    08 (mid-right)
03 (btm-left)    06 (btm-center)    09 (btm-right)
```

### 7-Point Grid (Landscape/Horizontal)
```
01 (top-left)    03 (top-center)    06 (top-right)
                 04 (center)
02 (btm-left)    05 (btm-center)    07 (btm-right)
```

### Placement Conventions
- Icon or Icon + Wordmark: typically top-left or top area
- Wordmark alone: typically bottom-left in large portrait layouts
- Maintain consistent corner/edge placement across a campaign

---

## Separating Logo and Symbol

When separating icon from wordmark:
- **Symbol:** flush to the left and top of the margins
- **Wordmark:** flush to the right and top of the margin
- When centered: keep natural alignment, each element flush with the margin

---

## Half-Width Placement

The wordmark can be used at half-width from margin to margin for an "impactful brand moment":
- Symbol placed top-left (flush to margin)
- Wordmark placed bottom, spanning approximately half the canvas width

---

## Co-Branding

- Always use the **full logo lockup** when co-branding
- Logos separated by **two units the size of the brandmark**
- One solid vertical stroke, **1.5pt width**, centered between logos
- Partner logo must **never be taller or wider** than the Vibrant logo
- Reasonable adjustments for optical balance are permitted

---

## Logo Restrictions (Never Do)

| Restriction |
|---|
| Never use the logo in any color other than black or white |
| Never add drop shadows, glows, or any effects |
| Never distort or stretch the logo |
| Never tilt, skew, or rotate the wordmark |
| Never outline the logo |
| Never use an unapproved format |
| Never use an incorrect version of the brandmark |
| Never scatter headline words across a layout (wordmark must always be intact) |
