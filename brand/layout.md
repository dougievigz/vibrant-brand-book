# Layout and Grid

## Margins

### Formula
**Shortest side of composition x 4% to 9%**

No exact science - use visual judgment within this range.

### Recommended Ranges
| Format | Margin |
|---|---|
| Portrait (tall) | 6% |
| Landscape (wide, smaller) | 9% |
| Landscape (wide, larger) | 5% |

### Rules
- Margins are clearspaces around edges ensuring no content bleeds off the page
- Too thin = content too close to edges (incorrect)
- Too thick = excessive wasted space (incorrect)
- Content must stay within margin boundaries

---

## Grid System

### Column Grid
- Approximately 4-5 evenly spaced vertical columns
- Dotted vertical grid lines (used as alignment guides)
- Grid lines extend full height of layout
- Elements (logo, headlines, body) snap to grid intersections

### Typography and Grid
- Typography aligns to and "hangs off" grid lines for visual interest
- Headlines can span multiple columns
- Body copy aligns to a single column or column group
- Logo placement follows the 9-point or 7-point grid systems (see `logo.md`)

---

## Typography Alignment in Layouts

### Left-Aligned (Primary)
- Most common setting for all typography
- Can appear large or full-width
- Can hang off an interesting grid line for visual interest
- Used for: all body copy, most headlines, most compositions

### Centered (Secondary)
- Only for communications where the brand wants to be very expressive
- Only for short headlines typeset with large copy
- NOT used for body copy or longer text blocks
- NOT used for the wordmark as a decorative centered element

---

## Photography in Layouts

- Photography occupies the dominant portion of layout area
- Text overlays photography when background allows legibility
- Logo/wordmark maintains consistent positioning (corners)
- Photography can be full-bleed within the layout frame
- White space at layout margins and between text blocks

---

## Layout Restrictions

Never do the following:

| Restriction |
|---|
| Do not scatter headline words across the layout in random positions |
| Do not rotate the wordmark vertically |
| Do not use the wordmark as an oversized decorative centered element |
| Do not use incorrect logo versions |
| Do not break the grid system |
| Do not let content bleed past margin boundaries |

---

## Iconography Grid

- **Construction grid:** 5x5
- **Style:** Simplified, slightly abstract representations
- **Construction rules:**
  - Mix of rounded and square edges
  - Line work always ends in straight edges
  - Always starts from one central dot

---

## Data Visualization

### Style
- Simplified line work illustration
- Dotted lines and solid lines
- Circular tabs (solid blue dots) at key data points
- Organic curved paths and concentric circles
- Used to break down complex information

### Colors
- Lines and dots: Reflex Blue (`#0142E2`)
- Background: light gray or black
- Text: Ink Black on light, White on dark

### Data Card Format (Dark Background)
- Dark semi-transparent or solid dark card
- Monospaced GT Pressura Mono for labels
- Example format:
  ```
  SHANNON'S INDEX
  Index Scale: 0-3
  Ref Range: +2.5
  Prev Value: 1.2
  Current Value: 2.3
  ```

## Ecosystem / Orbit Diagrams

- Central large dark circle (brand hub)
- Blue dashed circular orbit line
- Four satellite nodes at quadrant positions
- Circular photo thumbnails with white dashed borders
- Bold sans-serif for titles
- Monospaced all-caps for sub-items
- Background: solid black

---

## Journey / Hierarchy Diagrams

- Outer shape: elongated capsule/pill drawn with solid blue line
- Inner circles: concentric dashed blue circles, decreasing in size
- Connection points: blue filled dots
- Background: light warm gray
- Tier names: bold sans-serif
- Tier subtitles: all-caps monospaced small type
- Frequency labels: medium weight centered
